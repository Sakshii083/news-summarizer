# 🧠 AI News Summarizer

A Streamlit-based NLP web app that summarizes news articles using both rule-based and transformer models, with keyword extraction and reading-time insights.

---

## 🚀 Features
- ✂️ Automatic text summarization  
- 🔑 Keyword extraction  
- 🤖 AI-based summarization (Transformers - FLAN-T5)  
- 📊 Reading time comparison  
- 🌐 Interactive web UI using Streamlit  

---

## 🧠 Project Overview
This project is an NLP-based web application that generates concise summaries of news articles and extracts important keywords.

It supports two modes:
- Basic NLP summarization using NLTK  
- AI-powered summarization using Transformers  

---

## 🛠️ Tech Stack
- Python  
- NLTK  
- Transformers (Hugging Face)  
- Streamlit
